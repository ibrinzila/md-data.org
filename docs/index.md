# md-data.org docs

This repository provides a small FastAPI starter for Moldova public-data integrations.

## Endpoints

- `/status`
- `/v1/weather/current`
- `/v1/emergencies/alerts`
- `/v1/procurement/tenders`
- `/v1/procurement/tenders/{ocid}`
- `/v1/procurement/awards`
- `/v1/procurement/contracts`
- `/v1/procurement/budgets`
- `/v1/procurement/plans`
- `/v1/procurement/statistics`
- `/v1/statistics/summary`
- `/v1/finance/exchange-rates`
- `/v1/search`
- `/v1/datasets`
- `/v1/companies/search`
- `/v1/companies/{idno}`
- `/v1/ngos/search`
- `/v1/ngos/{ngo_id}`
- `/v1/legislation/search`
- `/v1/legislation/{edition_key}`
- `/v1/geospatial/layers`
- `/v1/geospatial/layers/{layer_key}`
- `/v1/geospatial/cadastre/search`
- `/v1/eu-funds/projects`
- `/v1/eu-funds/projects/{project_id}`
